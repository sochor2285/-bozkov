# Božkov Website

Moderní webová stránka pro značku Božkov rum vytvořená v Reactu.

## Funkce

- **Hero sekce** - Úvodní stránka s call-to-action tlačítkem
- **Produkty** - Přehled všech produktů Božkov včetně:
  - Božkov Original (tuzemský rum)
  - Božkov Republica Exclusive
  - Božkov Republica Solera
  - Božkov Republica Honey
  - Božkov Republica White
  - Božkov Republica Espresso
- **Historie** - Časová osa vývoje značky od roku 1948
- **Responzivní design** - Funguje na všech zařízeních
- **Moderní animace** - Plynulé přechody a efekty

## Instalace a spuštění

1. Nainstalujte závislosti:
```bash
npm install
```

2. Spusťte vývojový server:
```bash
npm run dev
```

3. Otevřete prohlížeč na adrese zobrazené v terminálu (obvykle http://localhost:5173)

## Build pro produkci

```bash
npm run build
```

Sestavené soubory budou v adresáři `dist/`.

## Technologie

- React 18
- Vite
- CSS3 s animacemi
- Responzivní grid layout

## Barvy a design

- Primární barva: Zlatá (#daa520) - symbolizuje kvalitu a tradici
- Sekundární: Tmavě hnědá (#2d1810) - připomíná barvu rumu
- Pozadí: Tmavé odstíny pro elegantní vzhled
